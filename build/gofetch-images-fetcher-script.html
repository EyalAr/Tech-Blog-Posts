GoFetch Images Fetcher Script

Friday, 16 November 2012
Eyal Arubas <eyalarubas@gmail.com>

'GoFetch' is an images fetcher script which I’ve written in Python.

It retrieves a list of images urls according to specified search parameters. The script can use either Bing’s or Google’s images search services. Search parameters can be defined via definitions ini file, or by prompting the user. The output is a list of image urls, which can be printed to stdout or to a file.

For Google’s service, an API key and a custom search engine ID are required.

For Bing’s service, an account ID is required, as well as a valid developer subscription to the search service.

For the source code, please visit this git repository.
